# Real Snobby Search Engine (RSSE)

Real S(nobby) Search Engine (RSSE) is a search engine project written in
Go. Its goal is to be the most crude implementation of a search engine,
doing the snobbiest job it can to find the information you need.

RSSE consists of a crawler which will scour the internet for content, and
a search appliance/UI where users can look for that content.

## What RSSE is not

RSEE is NOT a meta-search engine, meaning it does not aggregate results
from other search engines like Google, Bing, or Yandex. RSSE does its own
crawling and searching from scratch.

**Do not use this as a real search engine solution.** Consider a
meta-search engine solution like [SearXNG][sxng]. Meta-search engines are
able to use results from other search engines like Google, and aggregate
these results into your own private instance. They work pretty well. 😃
